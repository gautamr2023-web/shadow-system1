⚡ How Key Rotation Works
1. Oracle tries the active key first
2. If it hits a rate limit (429), it automatically switches to the next key
3. After all keys are tried, it waits and retries the first key
4. You can see which key is currently active in the list above
Adding 2–3 keys from different accounts gives you continuous uninterrupted access.
